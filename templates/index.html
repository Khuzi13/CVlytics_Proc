About The Project

This project was developed by an AI undergraduate student to bridge the gap between job seekers and Applicant Tracking Systems (ATS).

Using Python Django for the backend and React for the frontend, we utilize Machine Learning libraries to parse text, match keywords, and score resumes based on semantic similarity.
